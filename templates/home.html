Features of website

This website is made using html css and django only.

You can create new lists, access created lists or delete existing lists on this page.

You can create only one list for one list name.

The list-name cannot be blank.

You can click on name of list to view, add or delete tasks of that list.

You can login to admin panel by using following credentials

When you delete a list, all its tasks automatically get deleted. You can verify this by logging into the admin panel and deleting a list.

Also instead of database I have used Django Models for storing data.
