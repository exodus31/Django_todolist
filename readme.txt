Follow the steps to run the website:

Step 1 - Download the zip file and extract all its contents.
Step 2 - Download python from google.
Step 3 - Open command prompt and type - 'pip install django'. This command will install django in your pc.
Step 4 - Copy the local address where you have extracted the contents. Now type 'cd' and paste that local adress.Now type 'dir' in cmd. If you can see 'manage.py' file then you are at right place!!
Step 5 - Now type 'python manage.py runserver' in the cmd.You will be able to see a local link(usually starts with 'https://127') after running this command.
step 6 - Now go to chrome, type that local link and you should be able to see my website!!!
